# 分支语句

### 一、程序执行顺序
* 之前的程序都是按顺序从上到下执行的
* 程序执行的顺序可以是：顺序执行、分支执行、循环执行

### 二、if语句
* if

``` js

if(bool){
    //语句
}

```

* if...else

``` js

if(bool){
    //语句1
}else{
    //语句2
}

```
* if...else if...

``` js

if(bool1){
    //语句1
}else if(bool2){
    //语句2
}else{
    //语句3
}

```

### 三、switch
* 计算加减乘除

switch(n){
